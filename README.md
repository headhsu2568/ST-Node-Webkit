ST-Node-Webkit
==============

#### v0.1

A [node-webkit](https://github.com/rogerwang/node-webkit) Plugin for Sublime Text 2

<br />
Currently, I only test it at Sublime Text 2 on Windows XP and Windows 7

How to Install
--------------

* Be sure that you have installed [node-webkit](https://github.com/rogerwang/node-webkit)

* Set the path of Node-Webkit to `PATH` environment parameter
 
* Download this plugin, [ST-Node-Webkit](https://github.com/headhsu2568/ST-Node-Webkit/archive/master.zip)

* Open Sublime Text and choose `Menu → Preference → Browse Packages...`

* Extract the entire folder `NodeWebkit` to the folder opened above

* Restart Sublime Text

How to Use
----------
![screenshot](https://raw.github.com/headhsu2568/ST-Node-Webkit/master/screenshot.jpg)

* Load entire folder of a node-webkit application to Sublime Text (like `test` folder in the figure above)

* Open any file in the folder

* click `right mouse button → NodeWebkit`

* The application will be run through node-webkit !
